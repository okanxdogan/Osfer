# 🧠 Personal Productivity and Progress Tracking System

A modern, all-in-one personal productivity system designed to help you plan, track, and improve your daily life.

This project combines task management, focus tracking, reading tools, habit tracking, and personal analytics into a single, clean, and unified interface.

---

## 🚀 Overview

This application acts as a **personal life operating system**, allowing users to:

* Plan daily, weekly, and monthly tasks
* Track focus sessions with a timer
* Build habits using a visual chain system
* Read and annotate PDF books
* Monitor progress and personal statistics
* Manage everything from a single dashboard

---

## ✨ Features

### 🏠 Dashboard (You Page)

* Real-time personal stats
* Focus time tracking (in minutes)
* Active plans overview
* Habit chain summaries
* Clean and symmetrical UI layout

---

### 📅 General Plans

* Daily, Weekly, Monthly task management
* Add, edit, delete, complete tasks
* Internal scrolling task containers
* Clean and distraction-free UX

---

### 📚 Study System

* Create study blocks
* Add and manage study tasks
* Track progress per subject
* Fully customizable structure

---

### 📖 Reading System

* Upload and manage PDF books
* Open and read documents
* Page navigation with thumbnails
* Annotation tools:

  * Pen
  * Colors
  * Eraser
  * Notes
* Reading progress tracking (auto-saved)

---

### ⏱ Time (Focus System)

* Countdown timer (Focus / Break)
* Stopwatch mode
* Start / Pause / Reset controls
* Lap (session record) tracking
* Persistent timer state across navigation
* Fully integrated with dashboard focus stats

---

### 🔗 Don’t Break the Chain

* Create custom habit chains
* 365-day visual tracking system
* Daily interaction with notes
* Colorful completion feedback
* Multiple chain support

---

### ⚙️ Settings

* Theme customization (Dark / Light)
* User name personalization
* App behavior preferences
